## 一、标签定义（learnEngine/label.py）

买入价 = D+1 开盘价；卖出价 = D+1 收盘价（策略口径）

### 二分类标签

| 列名 | 计算公式 | 正样本条件 |
|------|---------|-----------|
| `label1` | (D+1收盘价 - D+1开盘价) / D+1开盘价 | ≥ 5% |
| `label1_3pct` | 同上 | ≥ 3% |
| `label1_8pct` | 同上 | ≥ 8% |
| `label2` | D+1阳线（收>开）且 D+2开盘 > D+1收盘 | 同时满足两条 |
| `label_d2_limit_down` | D+2当日涨跌幅（pct_chg） | ≤ -9.5%（D+2跌停） |
| `label_5d_30pct` | D+1~D+5 任意日的最高价 ≥ D0收盘价 × 1.30 | 达到即1 |
| `label_d1_open_up` | D+1开盘价 与 D0开盘价 比较 | D+1开盘 > D0开盘 |

### 浮点标签

| 列名 | 计算公式 |
|------|---------|
| `label_raw_return` | (D+1收盘价 - D+1开盘价) / D+1开盘价 |
| `label_open_gap` | (D+1开盘价 - D0收盘价) / D0收盘价（开盘溢价率） |
| `label_d1_high` | (D+1最高价 - D+1开盘价) / D+1开盘价（日内最大浮盈） |
| `label_d1_low` | (D+1最低价 - D+1开盘价) / D+1开盘价（日内最大回撤，负数） |
| `label_d1_pct_chg` | D+1 pct_chg（收盘相对前收涨跌幅%，与实盘统计口径对齐） |
| `label_d2_return` | (D+2收盘价 - D+1开盘价) / D+1开盘价（持有至D+2收盘的总收益） |

### 标签过滤规则
- D+1停牌（无数据）→ 整行跳过，不作负样本
- D+2无数据 → 涉及D+2的标签填 None
- D0无收盘价 → `label_open_gap` 填 None

### 2.1 板块热度（sector_heat）✅

**类型**：全局因子（无stock_code，广播到所有个股行）
**数据来源**：当日前5个交易日的涨幅榜板块数据

#### 输出因子

| 列名 | 当前计算口径 |
|------|---------|
| `adapt_score` | 板块轮动速度分，0~100，越高板块切换越快（主线越弱）。公式：相邻日板块重合率（权重80%）+ HHI集中度（权重20%），再乘主线强度修正系数（主线越强，最终分越低） |
| `adapt_score_norm` | `adapt_score / 100`，把轮动分归一化到 `[0,1]`，便于 XGBoost 直接切分快慢轮动区间 |
| `adapt_score_slow` | `1 - adapt_score_norm`，表示“慢轮动/主线持续”的环境强度 |
| `adapt_score_fast` | `adapt_score_norm`，表示“快轮动/题材切换快”的环境强度 |
| `adapt_score_mid` | `max(0, 1 - |adapt_score_norm-0.5|/0.5)`，表示“中等轮动” regime 强度 |

#### 本次口径对照

| 项目 | 上版本计算方法 | 你本次给的计算方法 | 我优化后的计算方法 |
|------|---------------|------------------|-------------------|
| `adapt_score` HHI 的 `total_seats` | 固定 `25`（5天×5板块） | 保持原口径，继续固定 `25`（你已做过实盘验证） | 已按你的要求恢复为固定 `25`，不再改成动态总席位 |
| `adapt_score` 提供形式 | 仅提供原始 `adapt_score` 单列 | 继续保留该轮动分，不改变定义 | 保留原始 `adapt_score`，并新增 `adapt_score_norm / slow / fast / mid`，让 XGBoost 更容易学习“快轮动 / 慢轮动 / 中轮动” regime |

> 说明：`adapt_score` 仍然是全局因子，同一天所有样本共享同一数值；它更适合作为**市场环境上下文**，与个股行为因子做交叉分裂，而不是单独承担横截面选股。

### 2.2 板块内个股特征（sector_stock）✅

**类型**：个股级因子
**数据来源**：不复权日线（kline_day）+ 分钟线（kline_min）
**时间窗口**：d0=当日，d1=昨日，d2=前日，d3=3日前，d4=4日前

#### 基本标识
| 列名 | 含义 |
|------|------|
| `sector_id` | 板块序号（1/2/3，对应当日Top3热板块的排名） |
| `sector_name` | 板块名称 |
| `stock_sector_20d_rank` | 个股在板块内近20日涨幅排名（1=最强，越小越强） |

#### 原始行情（d0~d4）
| 列名模式 | 含义 |
|---------|------|
| `stock_open_{d}` | 当日开盘价（不复权） |
| `stock_high_{d}` | 当日最高价（不复权） |
| `stock_low_{d}` | 当日最低价（不复权） |
| `stock_close_{d}` | 当日收盘价（不复权） |
| `stock_pct_chg_{d}` | 当日涨跌幅（%，来自kline_day.pct_chg） |

#### 量能因子（d0~d4）
| 列名模式 | 含义 |
|---------|------|
| `stock_amount_5d_ratio_{d}` | 当日成交额 / 近5日均成交额（放量>1，缩量<1，停牌=0） |
| `stock_vol_ratio_{d}` | 当日成交量 / 近5日均成交量（量比，停牌=0） |

#### 情绪合成分（d0~d4）
| 列名模式 | 含义 |
|---------|------|
| `stock_profit_{d}` | 上涨日的SEI值（多头情绪强度，0~100）；下跌/平盘日=0 |
| `stock_loss_{d}` | 下跌日的 100-SEI（空头情绪强度，越高越弱）；上涨/平盘日=0 |
| `stock_hdi_{d}` | HDI持股难度指数（0~100，越高持仓越煎熬，冲高回落时高） |

**SEI公式**：`base = 50 + 50×tanh(10×涨跌幅)` + 缺口调整(`gap×20`) + 趋势调整(`(R²-0.5)×15`) + VWAP惩罚(`-偏离×20`) + K线结构加减分(真阳+8/假阴+3/假阳-3/真阴-8) + 涨停封板/开板加减分

> ⚠️ **已知问题**：`gap_coeff=20` 过大，gap约±0.1×20=±2，加入后调整量可达±200分（SEI满分100），大量clip截断，严重压缩SEI在极端行情的区分度。建议降至5~8。

**HDI公式**：`(日内振幅×0.20 + 日内最大回撤×0.30 + 冲高回落幅度×0.20 + 涨跌幅绝对值×0.10 + VWAP穿越次数×0.10 + 量价背离率×0.10) × 100` + K线结构微调（±0.03~0.05）

#### 方向/价格原子因子（d0~d4）
| 列名模式 | 含义 |
|---------|------|
| `stock_gap_return_{d}` | 开盘缺口率 = (当日开盘 - 前收) / 前收；正=高开，负=低开 |
| `stock_candle_{d}` | K线结构：2=真阳(阳线且实涨), 1=假阳(阳线但实跌), 0=平盘/十字星, -1=假阴(阴线但实涨), -2=真阴(阴线且实跌) |
| `stock_cpr_{d}` | 收盘位置比 = (收盘价-当日最低) / (当日最高-当日最低)；0=收于最低，1=收于最高 |

#### 波动/持仓结构原子因子（d0~d4）
| 列名模式 | 含义 |
|---------|------|
| `stock_max_dd_{d}` | 日内最大回撤 = 分钟线从历史最高到当时收盘的最大跌幅（分钟线不可用时填0.10） |
| `stock_upper_shadow_{d}` | 上影线比率 = (当日最高 - max(开盘,收盘)) / 前收（上方抛压） |
| `stock_lower_shadow_{d}` | 下影线比率 = (min(开盘,收盘) - 当日最低) / 前收（下方支撑） |
| `stock_trend_r2_{d}` | 分钟线线性趋势R²（0=震荡无趋势，1=单边完美趋势；无分钟线填0.5） |
| `stock_vwap_dev_{d}` | VWAP偏离度 = 分钟线平均(|收盘-VWAP|/VWAP)（筹码散乱程度，无分钟线填0.01） |

#### 涨跌停行为因子（d0~d4）
| 列名模式 | 含义 |
|---------|------|
| `stock_seal_times_{d}` | 当日分钟线触及涨停价后再次回升的次数（封板次数） |
| `stock_break_times_{d}` | 当日分钟线触及涨停价后跌破涨停的次数（开板次数，越多情绪越弱） |
| `stock_lift_times_{d}` | 当日分钟线触及跌停价后反弹的次数（跌停翘板次数） |

#### 时间持续类因子（d0~d4）
| 列名模式 | 含义 |
|---------|------|
| `stock_red_time_ratio_{d}` | 红盘持续比 = 分钟线收盘>前收的分钟数 / 全天总分钟数（0=全天绿盘，1=全天红盘） |
| `stock_float_profit_time_ratio_{d}` | 浮盈持续比 = 分钟线收盘>昨日均价的分钟数 / 全天总分钟数（昨日均价=昨日成交额×1000/(昨日成交量×100)） |
| `stock_red_session_pm_ratio_{d}` | 红盘时段分布 = 午盘(13:00-15:00)红盘分钟数 / (早盘+午盘红盘分钟数)；无红盘=-1，早盘主导=0，均衡=0.5，午盘主导=1 |
| `stock_float_session_pm_ratio_{d}` | 浮盈时段分布（同red_session_pm_ratio逻辑，以昨日均价为基准） |

> ⚠️ **已知问题**：`red_session_pm_ratio=-1`（停牌标记）被当普通数值与[0,1]正常值混用，对XGBoost产生非单调影响。

#### 板块平均因子（d0~d4）
| 列名模式 | 含义 |
|---------|------|
| `sector_avg_profit_{d}` | 板块当日上涨股的SEI均值（赚钱效应强弱） |
| `sector_avg_loss_{d}` | 板块当日下跌股的(100-SEI)均值（亏钱效应强弱） |

### 2.3 均线位置（ma_position）✅

**类型**：个股级因子
**数据来源**：不复权日线收盘价（kline_day），近20个交易日
**口径**：统一不复权，与kline_day/分钟线/stk_factor_pro口径一致

| 列名 | 含义 |
|------|------|
| `ma5` | 近5个交易日收盘价均值（绝对价格，训练时通常排除） |
| `ma10` | 近10个交易日收盘价均值 |
| `ma13` | 近13个交易日收盘价均值 |
| `bias5` | (今日收盘 - MA5) / MA5 × 100（乖离率%，正=在均线上方） |
| `bias10` | (今日收盘 - MA10) / MA10 × 100 |
| `bias13` | (今日收盘 - MA13) / MA13 × 100 |
| `ma5_slope` | (今日MA5 - 昨日MA5) / 昨日MA5（短期动能方向，正=上行） |
| `ma_align` | 均线排列评分：2=MA5>MA10>MA13(完美多头)，1=弱多头，0=中性，-1=弱空头，-2=MA5<MA10<MA13(完美空头) |
| `pos_20d` | (今日收盘 - 近20日最低) / (近20日最高 - 近20日最低)（0=20日最低，1=20日最高） |
| `pos_5d` | (今日收盘 - 近5日最低) / (近5日最高 - 近5日最低)（近一周相对位置） |
| `from_high_20d` | (近20日最高 - 今日收盘) / 近20日最高（距20日高点的回撤幅度） |

### 2.5 市场宏观因子（market_macro）✅

**类型**：全局因子
**数据来源**：limit_list_ths（涨跌停池）/ limit_step（连板阶梯）/ limit_cpt_list（板块榜）/ index_daily（指数）/ kline_day（成交量）

| 列名 | 含义 |
|------|------|
| `market_limit_up_count` | 当日涨停池股票总数 |
| `market_limit_down_count` | 当日跌停池股票总数 |
| `market_max_consec_num` | 当日最高连板数（市场高度，反映资金集中度） |
| `market_consec_2plus_count` | 当日2板及以上股票数（连板梯队宽度） |
| `market_top_cpt_up_nums` | 当日最强板块（榜首）的涨停家数 |
| `market_top_cpt_cons_nums` | 当日最强板块（榜首）的连板家数 |
| `index_sh_pct_chg` | 当日上证指数涨跌幅（%） |
| `index_sz_pct_chg` | 当日深证成指涨跌幅（%） |
| `index_cyb_pct_chg` | 当日创业板指涨跌幅（%） |
| `market_vol_ratio_d0` | 当日全市场总成交额 / 近4日（D-1~D-4）均成交额（**d0 分母不含自身**） |
| `market_vol_ratio_d1~d4` | D-1~D-4全市场总成交额 / 近5日全部均成交额 |
| `market_amount_d0` | 当日全市场总成交额（单位：万亿元） |
| `market_amount_5d_avg` | 近4日（D-1~D-4）全市场总成交额均值（万亿元） |
| `market_amount_d0_vs_5d` | market_amount_d0 / market_amount_5d_avg（当日相对近期活跃度） |
| `market_amt30_count` | 当日成交额 ≥ 30亿的股票数 |
| `market_amt30_above5d_count` | 当日成交额 ≥ 30亿 且高于自身近5日均成交额的股票数 |
| `market_amt30_above5d_newlow5_count` | 上述活跃放量股票中，收盘创近5日新低的股票数 |
| `market_amt30_above5d_holf_count` | 上述活跃放量股票中，高开低走（open>pre_close 且 close<open）的股票数 |
| `market_amt30_above5d_lohc_count` | 上述活跃放量股票中，低开高走（open<pre_close 且 close>pre_close）的股票数 |
| `market_up_60pct_board_count` | 当日涨幅达到各自涨跌停板幅度 60% 的股票数（强势广度） |
| `market_down_60pct_board_count` | 当日跌幅达到各自涨跌停板幅度 60% 的股票数（弱势广度） |
| `market_limit_up_rate` | 当日涨停数 / 5200（全市场总股数近似）≈ 涨停参与率 |
| `market_limit_up_5d_trend` | 当日涨停数 / 近4日均涨停数（涨停趋势，clip[0.1, 10.0]） |
| `market_consec_5d_trend` | 当日最高连板数 / 近4日均最高连板数（连板趋势，clip[0.1, 10.0]） |

#### 本次口径对照

| 项目 | 上版本计算方法 | 你本次给的计算方法 | 我优化后的计算方法 |
|------|---------------|------------------|-------------------|
| `market_vol_ratio_d0` | `vol_d0 / mean(vol_d0~d4)`，分子参与分母，自引用 | 你要求从“盘面变化”角度严谨表达市场活跃度 | 改为 `vol_d0 / mean(vol_d1~d4)`，只和历史比较，避免把当天自己稀释掉 |

**类型**：个股级因子

#### 振幅类（d0~d4）

### 2.13 涨停板情绪因子（limit_emotion）✅

**新增日期**：2026-04-01
**类型**：全局因子（无stock_code，广播到所有个股行）
**数据来源**：`limit_list_ths` 表（同花顺口径，`limit_type="炸板池"`/`"涨停池"`）+ 分钟线（`kline_min`）

#### 输出因子

| 列名 | 含义 | 单位 | 中性值 |
|------|------|------|-------|
| `market_zhaban_count` | D0 炸板数（当日最高价触及涨停、收盘未封板的股票数，同花顺口径） | 只 | 0 |
| `market_zhaban_ratio` | D0炸板数 / mean(D1~D4炸板数)，clip[0.1, 10.0] | 倍 | 1.0 |
| `market_limit_touch_amount` | D0 涨停+炸板股中，首次触板那根分钟K线成交额全市场累加（剔除ST） | 万亿元 | 0.0 |
| `market_limit_touch_amount_ratio` | D0触板分钟成交额 / mean(D1~D4触板分钟成交额)，clip[0.1, 10.0] | 倍 | 1.0 |

#### 计算口径

**炸板数**：直接查 `limit_list_ths` 表 `limit_type="炸板池"`，与涨停数口径统一（同一数据源）。

**触板分钟成交额（精确定义）**：
1. 取当日涨停池 + 炸板池所有股票，剔除 ST（ST 涨幅5%，干扰主板/创业板信号）
2. 查日线 `high` 作为涨停价参照（触板股当日 `high = 涨停价`）
3. 对每只股票，加载 D 日分钟线，找首次 `bar.high >= daily_high - 0.01`（0.01元容差防浮点误差）的那根K线
4. 累加该根K线的 `amount`（Tushare stk_mins 原始单位为**元**，代码中 `÷1000` 转换为千元后再累加，与日线成交额口径一致）

**D1~D4历史分母**：同样用分钟线精确触板成交额（`data_bundle._load_limit_touch_data` 并发加载D0~D4全部），分子分母口径完全一致。

#### 三维联动解读（配合 market_macro 的 market_limit_up_5d_trend 使用）

| 信号组合 | 市场含义 |
|---------|---------|
| 涨停多 + 炸板少 + 触板成交高 | 资金积极封板，做多合力强 |
| 涨停多 + 炸板多 + 触板成交高 | 多空分歧大，博弈激烈但封板意愿不足 |
| 涨停少 + 炸板多 + 触板成交低 | 情绪收敛，空方主导 |

#### IO 设计

- `macro_cache["zhaban_df"]`：D0炸板池 DataFrame
- `macro_cache["zhaban_counts_5d"]`：`{date: count}` D0~D4每日炸板数
- `macro_cache["limit_touch_codes_5d"]`：`{date: [ts_code,...]}` D0~D4每日涨停+炸板codes
- `macro_cache["limit_touch_amount_5d"]`：`{date: amount_千元}` 分钟线精确触板成交额
- 因子 `calculate()` 内零 IO，全部从 `macro_cache` 读取
